# SyrupBTC Overview

SyrupBTC is Maple's Bitcoin on-ramp for institutional liquidity. The product extends Maple Pools with a controller and router layer so wBTC deposit assets can be wrapped into the pool asset, routed through Maple-managed strategies, and unwound back to wBTC when lenders exit.
